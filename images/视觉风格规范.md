## 一、整体布局

### 1.1 三栏布局结构

```
─────────────────────────────────────────────────────────────────────
│  标题栏（macOS 风格窗口控制按钮 + 面包屑导航 + 操作按钮）              │
├──────────┬──────────────────────────────────┬───────────────────────┤
│          │                                  │                       │
│  左侧栏   │          中间主区域               │       右侧栏          │
│  240px   │         自适应剩余空间             │       320px          │
│  固定宽度  │                                  │       固定宽度        │
│          │                                  │                       │
│  · Logo   │  · 文档列表区（上半部分）          │  · AI 助手面板        │
│  · 新建   │  · 编辑器主区域（下半部分）        │  · 文档大纲 TOC       │
│  · 导航   │                                  │                       │
│  · 知识库  │                                  │                       │
│  · 用户   │                                  │                       │
```

### 1.2 布局参数

| 区域 | 宽度 | 说明 |
|------|------|------|

| 中间主区域 | flex: 1 | 自适应剩余空间 |

## 八、动效与过渡

### 8.1 过渡时长

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮 hover | 150ms | `ease-out` |
| 面板展开/收起 | 250ms | `ease-in-out` |
| 侧边栏折叠 | 300ms | `ease-in-out` |
| 列表项出现 | 200ms | `ease-out` |
| 消息气泡出现 | 200ms | `ease-out` |
| 页面切换 | 250ms | `ease-in-out` |

### 8.2 特殊动效

- **AI 助手头像**：蓝色光晕脉冲动画（`box-shadow` 呼吸效果）
- **流式输出**：光标闪烁动画
- **加载状态**：骨架屏（shimmer 效果），背景渐变动画
- **文档保存**：顶部短暂绿色提示条，1.5s 后消失

## 十、暗色模式说明

当前版本仅支持暗色主题（Dark Mode），不提供亮色模式切换。

所有色值均基于暗色主题设计，后续如需支持亮色模式，需重新定义色彩 Token。

## 十二、设计原则总结

1. **深色沉浸**：整体深蓝灰色调，减少视觉疲劳，适合长时间阅读和写作
2. **层次分明**：通过色彩明度区分信息层级，重要内容更亮，辅助信息更暗
3. **蓝色引导**：蓝色作为主强调色，引导用户注意力到关键交互元素
4. **圆角柔和**：统一的圆角系统，营造现代、友好的视觉感受
5. **留白呼吸**：充足的间距让内容不拥挤，编辑器区域最大宽度 720px 保证阅读舒适度
6. **微动效**：克制的过渡动画提升交互质感，不过度花哨
